# Ecommerce-SQL-Analysis
Built an e-commerce database in MySQL and performed SQL analysis to uncover insights on revenue by location, repeat customers, and product category performance.

This project showcases SQL database design and business analysis using an e-commerce dataset. I created and populated the Customers, Orders, 
OrderDetails, and Products tables with sample data. I then performed SQL-based analyses to identify revenue contribution by location, percentage 
of repeat customers,top-selling product category, and revenue generated by each product category, providing actionable business insights through 
optimized SQL queries.
